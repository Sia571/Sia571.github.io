# 张思辰 / Sia Zhang — 个人作品集

一个零依赖、响应式的中文个人作品集网站，采用独立杂志与品牌 Editorial 的排版方式。主要内容包括：

- 个人定位、教育背景与专业主线
- 五个职业与个人切面入口
- 四篇经过脱敏、可独立展开与收起的专业案例文章
- 五个入口统一的“查看—返回目录—已读”阅读状态
- 常驻的十章阅读路线，覆盖开场、关于、五个 Sia、案例、AI 实践、经历索引、现场、生活、联系与留言
- 章节末尾的“下一章”衔接，以及阅读过程中可见的“返回五个 Sia 目录”入口
- “165 个问题如何形成 11 类知识结构”的抽象交互
- AIGC 变量关系示意与 ToB 项目数字墙（均不冒充比例统计图）
- UCL 六篇课程作业所形成的健康人文问题地图
- 随机抽取的“非简历小纸条”互动
- 网站制作过程的 AI 协作案例
- 只记录已确认内容的经历索引
- 分布在对应案例中的专业现场照片，以及牛津会议旁听、佛罗伦萨 Plan B、旅行档案与长期健康改变
- 支持本地预览与 Supabase 云端共享两种模式的留言簿；公开留言会自动轮播
- 对儿童、家庭、内部材料与医学信息的公开边界说明
- 公开联系方式与按岗位提供简历的说明

## 本地预览

在本文件夹打开终端，运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

网站不使用第三方统计、追踪、远程字体或外部图片。所有照片均为适配网页的副本，新加入的公开图片已移除拍摄元数据；不公开儿童与服务对象面孔、真实患者资料或公司内部材料。公开案例采用重新叙述与脱敏方式，未确认的职责、结果与数字不做推导补写。

留言簿未填写 `config.js` 时使用浏览器本地存储；填写 Supabase Project URL 和 Publishable key 后自动切换为云端共享模式。数据库结构、RLS 权限、审核方式与 GitHub Pages 发布步骤见 `PUBLISHING.md`。

UCL School of Pharmacy 照片保留原始横屏现场，以同时呈现本人、学院门牌及入口；画面中的路人属于原始公共场景。

V6 移除了阅读路线在滚动时自动牵引导航项的动作。路线仍会被动标记当前章节，但不会再控制页面位置；药学院横屏照片在桌面端回到文字右侧完整展示。

健康人文课程区采用两层展开：先打开六篇作业目录，再单独展开某一篇的真实英文题目和中文摘要；同一时间只显示一篇。前五篇摘要依据提交版课程论文整理，第六篇依据 2026 年 7 月 21 日的硕士论文修订草稿整理，并明确标注为研究进行中。

AIGC 用户研究案例列出已发表论文的题目、刊期与核心结论。论文 PDF 作为网站文件随页面一同部署，避免期刊服务器链接失效；外部仅保留 DOI 永久链接，两个入口均在新标签页打开。
